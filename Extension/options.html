In order to function this extension asks for the following permissions:

  • Access to read all pages

    We search each page you visit for consent-related popups that we know how to handle

  • Information about tab URLs

    You can turn the extension on/off on a page-by-page basis by clicking the icon. To check if it is enabled it needs to know the address

  • Storage

    Your preferences and settings are stored directly in your browser

Without these permissions it will simply do nothing.
